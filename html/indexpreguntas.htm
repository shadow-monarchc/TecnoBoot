¿cuando se creo el desarrollador web?

La fase inicial de la web, conocida como Web 1.0 o web estática, abarcó desde 1991 hasta 2003. Durante este período, las páginas web eran principalmente estáticas y la interacción entre el usuario y el sitio web era bastante limitada. Estos primeros días de la web se caracterizaron por su simplicidad y velocidad
